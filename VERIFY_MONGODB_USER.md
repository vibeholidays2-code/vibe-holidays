## Common Issues

### Password Has Special Characters
If your password has `@`, `#`, `$`, etc., they need to be URL encoded:
- `@` ‚Üí `%40`
- `#` ‚Üí `%23`
- `$` ‚Üí `%24`

### User Permissions Wrong
User must have "Read and write to any database" or "Atlas admin"

### IP Not Whitelisted
Must have 0.0.0.0/0 in Network Access
